# Transport Profit Tracker

A lightweight Expo + TypeScript app to track daily transport income, expenses, and profit.

Quick start:

1. cd transport-profit-tracker
2. npm install
3. Create a Supabase project and set environment variables in a .env or through Expo config:
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
4. npm start

This scaffold includes:
- Expo + TypeScript
- React Navigation
- React Native Paper (UI)
- NativeWind (Tailwind)
- Victory Native for charts
- Supabase client placeholder
